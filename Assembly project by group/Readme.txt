General Usage Notes
-------------------

- The program asks user to input value for the size of stack.

- Once a valid n is input, the program will let user input elements.

- After all of the integers input, the program will print out the sorted stack list

- Then the program will ask user to input a number to search from the stack list

- The program will print out messages for whether the input is found or not.
